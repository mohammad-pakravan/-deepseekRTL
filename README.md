# 🇮🇷 Persian Chat RTL Fixer

افزونه کروم برای راست‌چین کردن خودکار پیام‌های فارسی در سایت چت

 

## ✨ ویژگی‌ها

- ✅ راست‌چین کردن خودکار پیام‌های فارسی
- ✅ فعال/غیرفعال کردن با یک کلیک
- ✅ تنظیم سایز فونت (ریز، متوسط، بزرگ، خیلی بزرگ)
- ✅ پشتیبانی از لیست‌ها، هدینگ‌ها و جدول‌ها
- ✅ نادیده گرفتن بلاک کدها و دکمه‌ها
- ✅ ذخیره خودکار تنظیمات

## 📥 نصب

### روش 1: نصب دستی (برای همه)
1. فایل `persian-rtl-fixer-v1.0.zip` رو دانلود و اکسترکت کن
2. در کروم به آدرس `chrome://extensions` برو
3. **حالت توسعه‌دهنده** رو روشن کن
4. روی **Load unpacked** کلیک کن و پوشه اکسترکت شده رو انتخاب کن
